#### Django 4.x Cheatsheet

##### Pre-Requisites for django
##### guidance for Windows
* Install Python
* Install Pip
* Install Pipenv 
* Create a Django base directory for all your django projects (e.g., d:\DJANGO)
* Create an Env folder like env_myproject1 in that django base dir (e.g., d:\DJANGO\env_django1)
* Create Virtual Env
  * Go to your Django base directory
  * Go to your Django environment directry
  * run pipenv shell in this directory (first time creates, next time launches)
* Install your django and other project creation work in this environment 
* Launch VS code via code . from this folder
